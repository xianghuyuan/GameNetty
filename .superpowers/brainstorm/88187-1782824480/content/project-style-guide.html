GameNetty 当前项目风格锚点

以现有 BattleMainWindow 正式参考图和 UI_BattleMain_ArtSpec 为准。后续大厅、章节、构筑、养成界面都应从这里延展。

应该保持

核心气质

中等饱和、奶油月夜、轻幻想、横版推进、低疲劳、清晰可读。

UI 材质

香草灰/奶油色圆角面板，浅金细边，哑光低高光，柔和蓝紫阴影。

点缀色

浅粉/玫瑰红用于生命和战斗压力，薄荷绿用于能量和成长，淡天蓝用于风/冷却，浅金用于边框和奖励。

背景

蓝紫月夜森林，低到中对比，横向可延展。背景让位给 HUD 和角色，不做强中心光斑。

必须避免

不要厚重 RPG

不要皮革、铆钉、暗木、重金属、蜡封、哥特、脏旧羊皮纸。

不要纸雕/纸剧场

不要明显纸纤维、厚纸板边、舞台幕布、吊绳、剧场框。

不要科技霓虹

不要黑底赛博、工业机械、现代 app 面板、强荧光高对比。

不要过度可爱或甜品化

可以奶油色，但不是糖果店、甜点 UI、婴儿风。

对 1280×720 横屏局外界面的推导

A

推荐方向：奶油月夜轻幻想局外 UI

大厅、章节、构筑、养成都使用柔和圆角面板和蓝紫月夜场景；信息密度比战斗 HUD 高，但仍保持低疲劳。

B

不推荐：纸雕舞台化

之前提到的纸雕感需要收回，项目规格明确要求不要纸板、纸剧场、纸纤维。

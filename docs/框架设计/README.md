## 🎯 框架设计特点

### 通用性强
框架层的设计具有高度的通用性和可复用性，可以支撑多种不同的游戏玩法。

### 影响范围大
框架层的修改会影响到所有基于它的功能模块，因此需要谨慎设计和充分测试。

### 需要深入理解
开发框架层需要深入理解 ET 框架的核心概念：
- Entity-Component-System (ECS) 架构
- Fiber 单线程模型
- IScene 场景继承链
- EntityRef 引用机制
- 事件系统

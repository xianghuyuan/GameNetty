# 时差滚动系统实现文档 (Parallax Scrolling System)

## 1. 概述
在 2D 游戏中，时差滚动通过让背景层以不同的速率移动，营造出深度感。GameNetty 采用 **ECS + View 绑定** 的方式实现该功能。

---

## 2. 核心组件 (Model 层)
定义于 `Model/Component/ParallaxComponent.cs`。

```csharp
[ComponentOf(typeof(Unit))] // 背景也是一个特殊的 Unit
public class ParallaxComponent : Entity, IAwake<Vector2>, IUpdate
{
    public Vector2 Multiplier;       // 滚动系数 (0:静止, 1:随相机同步)
    public Vector3 StartPosition;    // 逻辑初始位置
}

[ComponentOf(typeof(Unit))]
public class ParallaxViewComponent : Entity, IAwake
{
    public Transform Transform;      // 对应的 Unity Transform
    public float TextureWidth;       // 贴图宽度，用于无限循环
}
```

---

## 3. 逻辑系统 (Hotfix 层)
定义于 `Hotfix/System/ParallaxComponentSystem.cs`。

```csharp
[EntitySystemOf(typeof(ParallaxComponent))]
public static partial class ParallaxComponentSystem
{
    [EntitySystem]
    private static void Update(this ParallaxComponent self)
    {
        // 1. 获取主相机坐标 (逻辑相机)
        Vector3 cameraPos = GlobalComponent.Instance.MainCameraPos;

        // 2. 计算基于时差系数的位移
        float distX = cameraPos.x * self.Multiplier.x;
        float distY = cameraPos.y * self.Multiplier.y;

        // 3. 更新 View 层 Transform
        var view = self.GetParent<Unit>().GetComponent<ParallaxViewComponent>();
        view.Transform.position = new Vector3(
            self.StartPosition.x + distX, 
            self.StartPosition.y + distY, 
            0);

        // 4. 处理无限循环逻辑 (可选)
        float tempX = cameraPos.x * (1 - self.Multiplier.x);
        if (tempX > self.StartPosition.x + view.TextureWidth) {
            self.StartPosition.x += view.TextureWidth;
        }
    }
}
```

## 4. 配表集成 (Luban)
开发者应在 `Excel/GameConfig/BackgroundConfig.xlsx` 中配置背景层级。

| Id | PrefabPath | MultiplierX | MultiplierY |
| :--- | :--- | :--- | :--- |
| 2001 | Prefab/BG/Sky | 0.95 | 0.95 |
| 2002 | Prefab/BG/Mountain | 0.60 | 0.60 |

---

## 5. 使用流程
1. **创建 Entity**: 在场景初始化时，根据配置创建背景 Unit。
2. **挂载组件**: 调用 `unit.AddComponent<ParallaxComponent>(config.Multiplier)`。
3. **视图绑定**: 在 `HotfixView` 层通过 `TUIForm` 加载资源并绑定 `Transform` 到 `ParallaxViewComponent`。

---

## 6. 性能优化
- **Update 节流**: 对于极远景，可以降低 Update 频率。
- **GPU 方案**: 对于纯纹理背景，建议直接使用 Shader 修改材质 Offset。

## 当前实现关键结论

### 服务端职责边界

- `BattleAIComponentSystem.Update` 负责自动战斗总入口。
- `BattleActionDecisionComponentSystem.Update` 负责选目标、选技能、判定追击 / 攻击状态。
- `BattleMoveComponentSystem` 负责固定点移动、FollowTarget 跟随、停止移动与移动广播。
- `BattleSkillHelper` 负责技能选择、施法校验、目标筛选、效果执行与伤害计算。
- `BattleUnitHelper` 负责把服务端战斗结果广播到客户端。

### 客户端职责边界

- 消息 Handler 负责接收服务端同步并更新本地战斗实体。
- `BattleMessageLogHelper` 负责输出“收到消息注释”类日志，便于定位链路。
- `BattleUnitViewComponent` 负责攻击反馈、受击反馈等表现。
- `BattleUIHelper` / `BattleHUDWindow` 负责控制模式与单位 HUD 展示。

# 战斗主界面资源需求

本文档只描述 `BattleMainWindow` 的美术资源和 Prefab 接入要求。通用迭代规则见 [spec/art/README.md](./README.md)。

## 目标

- 生成战斗主界面 HUD 所需 PNG 资源。
- 界面效果图和 Prefab 布局坐标按 `1280x720` 画布制作。
- 支持玩家血量/能量、Boss 血量、波次、系统按钮、发射器栏和冷却显示。
- UI 层级固定在 Prefab 中，业务代码只刷新数据和响应点击。

## 正式路径

```text
画布: 1280x720
效果图: spec/art/ui-reference-atlas/BattleMainWindow/source/BattleMainWindow.png
参考图集: spec/art/ui-reference-atlas/BattleMainWindow/atlas/BattleMainWindow_atlas.png
拆分图: spec/art/ui-reference-atlas/BattleMainWindow/sprites/
Prefab: Unity/Assets/AssetRaw/UI/Battle/BattleMainWindow.prefab
小图: Unity/Assets/AssetRaw/UIRaw/Atlas/Battle/battle_main/
```

规则：

- `ui-reference-atlas/BattleMainWindow/` 只保留当前确认版本。
- `source/BattleMainWindow.png` 是当前界面效果图，替换时保持同名。
- `battle_main/` 只保留当前 Prefab 实际引用的 PNG。
- 不再引用的 PNG 必须删除。
- 不使用 `_Incoming`、`Raw/Generated` 或临时目录作为正式资源来源。

## Prefab 绑定

根节点：

- `BattleMainWindow`
- 必须挂 `GameLogic.UIBindComponent`

绑定字段：

| 字段 | 类型 | 用途 |
| --- | --- | --- |
| `m_imgPlayerHp` | `Image` | 玩家血量进度 |
| `m_tmpPlayerHp` | `GameLogic.UIText` | 玩家血量文本 |
| `m_imgPlayerEnergy` | `Image` | 玩家能量进度 |
| `m_tmpControlMode` | `GameLogic.UIText` | 控制模式文本 |
| `m_imgBossHp` | `Image` | Boss 血量进度 |
| `m_tmpBossName` | `GameLogic.UIText` | Boss 名称 |
| `m_tmpBossHp` | `GameLogic.UIText` | Boss 血量文本 |
| `m_tmpWave` | `GameLogic.UIText` | 波次文本 |
| `m_tfEmitterBar` | `Transform` | 发射器栏容器 |
| `m_itemEmitterSlot` | `GameObject` | 发射器槽模板 |
| `m_imgCooldown05` | `Image` | 冷却圆形进度 |
| `m_tmpCooldown05` | `GameLogic.UIText` | 冷却秒数 |
| `m_btnPause` | `GameLogic.UIButton` | 暂停 |
| `m_btnBookmark` | `GameLogic.UIButton` | 书签/图鉴 |
| `m_btnGear` | `GameLogic.UIButton` | 添加/调试 |

## 运行时规则

- 血条和能量条左侧固定，通过宽度或 `fillAmount` 表示进度。
- Boss 血条、玩家血条、玩家能量条不得使用半格截图作为满格资源。
- 冷却底盘和冷却进度必须是两张 PNG，不使用合成图。
- 冷却进度使用 `Filled / Radial360`。
- 点击节点使用 `GameLogic.UIButton`，事件通过 `UIButton.SetClick(...)` 注册。
- 不在业务代码中动态创建 UI 层级。

## 非目标

- 普通杂兵 HUD 血条。
- 多管 Boss 血条。
- 经验条、金币面板、背包格、装备槽。
- 业务逻辑、协议和配置表。

## 验收

- 正式目录没有旧版本、预览图或未引用 PNG。
- Prefab 截图与当前确认参考图集的主体资源一致。
- 文本使用 `GameLogic.UIText`。
- 按钮使用 `GameLogic.UIButton`。
- Unity 编译无绑定字段缺失。

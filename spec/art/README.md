# 美术需求文档

本目录存放美术资源规格，负责说明资源用途、命名、目录归属、拆分要求和接入约束。

## 迭代共识

- 默认先预览，用户确认后才保存为当前正式版本。
- UI 界面效果图默认画布为 `1280x720`，除非单个界面文档明确写出其他尺寸。
- 同一功能、同一界面、同一类产物只保留一份当前版本。
- 不用 `preview`、`effect`、`v2`、`Generated`、`Detailed`、`Split`、`Painterly` 等后缀或目录长期保存尝试版本。
- 确认新版本后，替换当前文件，并删除旧的无引用产物。
- 用户明确手动拼图时，不生成或保留 atlas 坐标清单/json。
- 缩放拆分 PNG 时必须保持原始宽高比；布局适配通过 Prefab 的 RectTransform、Image Type 或 9-slice 完成，不直接拉伸源图。
- 临时产物不得进入 Unity 正式资源目录。

推荐目录语义：

```text
spec/art/<feature>/
  source/   # 当前确认的源图，可选
  sprites/  # 当前确认的拆分 PNG
  atlas/    # 当前确认的透明图集 PNG
```

这些目录表示当前版本，不做历史归档。
